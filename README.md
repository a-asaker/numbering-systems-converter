# Numbering Systems Converter

 Number System Converter Is A Ruby Project Which Converts A Number To Different Numbering Systems :
 
     -Binary To Decimal
     -Decimal To Binary   
     -Decimal To Octal    
     -Octal To Decimal    
     -Decimal To Hexa-decimal    
     -Hexa-decimal To Decimal
